# FirstLayer

**Learn today. Lead tomorrow.**

FirstLayer is a platform designed to help aspiring entrepreneurs start their first business from zero.
The goal of the project is to provide clear, practical guidance for building a business step-by-step and documenting the journey of creating a modern startup.

This repository contains the source code for the **FirstLayer website**.

---

## 🚀 Vision

Starting a business can feel overwhelming. FirstLayer aims to simplify the process by breaking entrepreneurship into clear, actionable steps.

The platform focuses on:

* Learning the fundamentals of business
* Launching your first venture
* Building systems that generate long-term value
* Documenting the process publicly

Over time, FirstLayer will expand into a broader ecosystem including education, tools, and resources for entrepreneurs.

---

## 🛠 Tech Stack

The FirstLayer website is built using modern web technologies:

* **Next.js** – Full-stack React framework
* **React** – UI library
* **Tailwind CSS** – Utility-first styling
* **TypeScript** – Type safety for scalable development
* **Vercel** – Deployment and hosting

Future integrations may include:

* PostgreSQL
* Prisma ORM
* Authentication services
* Analytics and growth tools

## ⚡ Getting Started

To run the project locally:

### 1. Clone the repository

```
git clone https://github.com/YOUR-USERNAME/firstlayer.git
```

### 2. Navigate into the project directory

```
cd firstlayer
```

### 3. Install dependencies

```
npm install
```

### 4. Start the development server

```
npm run dev
```

Then open:

```
http://localhost:3000
```
